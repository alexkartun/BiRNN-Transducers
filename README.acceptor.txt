How to run experiment.py file?

Type:
    python3 experiment.py train_filename, dev_filename, type_of_data_set

Should be 3 parameters to the script.
1) train_filename - train's filename.
2) dev_filename - dev's filename.
3) type_of_data_set - name of the directory where the files are located

   F.E: python3 experiment.py train dev data
   * 'data': directory that located near the experiment.py file
   * train, dev: files located in the 'data' directory
